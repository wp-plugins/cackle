This plugin integrates "Cackle" social commenting system (Facebook, Vkontakte, Mail.ru, e.t.c.) right into your website. 

== Description ==

This plugin allows users to use social networks accounts to leave comments.

== Installation ==

1. Upload  folder `cackle` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
